cmd에서 root로 mysql 접속한 다음 아래 과정 진행해주세요!

1. exit 유저 생성
create user 'exit'@localhost identified by 'exit';

2. exit 테이블 생성
create database exit default character set utf8;

3. 권한 설정
grant all privileges on exit.* to 'exit'@localhost with grant option;

4. 1-3번까지 적용
flush privileges;

5. 이클립스, dbeaver에 DB연결
